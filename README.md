# misswhale-ui

Vue3 + Nuxt3 UI Components Library Test

## Install

```
# npm
npm install @misswhale/ui
npm install --save-dev @iconify/vue sass

# yarn
yarn add @misswhale/ui
yarn add -D @iconify/vue sass
```

## Nuxt3 Import

```
export default defineNuxtConfig({
  modules: [
    "@misswhale/ui/nuxt",
    // other modules
  ],
});
```
